# Syntecxhub Expense Tracker

A responsive expense tracking application built with React as part of the Syntecxhub Web Development Internship Week 1 Task.

## Features

- ✅ Add, edit, and delete expenses
- ✅ Real-time expense tracking
- ✅ Data fetching from mock API
- ✅ Performance optimized with useMemo and useCallback
- ✅ Responsive design
- ✅ Form field focus management with useRef

## Technologies Used

- React 18
- Vite
- Modern React Hooks (useState, useEffect, useRef, useMemo, useCallback)
- CSS3

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
